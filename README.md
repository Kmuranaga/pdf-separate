# PDF分割ツール

PDFファイルを1ページずつに分割するシンプルなWebツールです。

## 機能

- PDFを1ページずつ個別のPDFファイルに分割
- 個別ダウンロードまたはZIPで一括ダウンロード
- ブラウザ上で完結（サーバーへのアップロード不要）
- ドラッグ＆ドロップ対応

## 使い方

### 方法1: ローカルサーバーで起動（推奨）

```bash
# Python 3の場合
cd pdf-splitter
python -m http.server 8000

# ブラウザで開く
# http://localhost:8000
```

### 方法2: ファイルを直接開く

`index.html` をブラウザにドラッグ＆ドロップ、またはダブルクリックで開く

> ※ 一部ブラウザではセキュリティ制限で動作しない場合があります

## 操作手順

1. PDFファイルをドラッグ＆ドロップ（またはクリックして選択）
2. 「PDFを分割する」ボタンをクリック
3. 分割完了後、個別またはZIPでダウンロード

## 出力ファイル名

```
元ファイル名_001.pdf
元ファイル名_002.pdf
元ファイル名_003.pdf
...
```

## 技術仕様

- **pdf-lib**: PDF分割処理
- **JSZip**: ZIP圧縮
- **FileSaver.js**: ファイルダウンロード

すべての処理はブラウザ内で行われ、ファイルは外部サーバーに送信されません。

## 動作環境

- Chrome（推奨）
- Firefox
- Safari
- Edge
